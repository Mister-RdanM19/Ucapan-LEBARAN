# 🌙 EID MUBARAK PROJECT: MR.RM19 TO N3I

![Author](https://img.shields.io/badge/Author-Mr.Rm19-blue?)

## 📌 Deskripsi Proyek
Proyek ini adalah kumpulan 20 halaman web statis (HTML/CSS/JS) yang berisi ucapan Selamat Idul Fitri dari **Mr.Rm19** khusus untuk **N3i**. Setiap file dirancang dengan desain visual yang berbeda-beda, mulai dari tema *Luxury Gold*, *Cyberpunk Neon*, hingga *Minimalist Elegant*.

Proyek ini dibuat untuk berjalan secara ringan di **GitHub Pages**, memungkinkan pengiriman pesan yang unik dan personal setiap harinya selama nuansa Lebaran.

##  Struktur File
Setiap file memiliki logika dan desain visualnya sendiri:
* `ucapanlebaran-1.html` - `ucapanlebaran-5.html`: Tema Glassmorphism & Sacred Aurora.
* `ucapanlebaran-6.html` - `ucapanlebaran-10.html`: Fokus pada elemen Ketupat & Glitch Typography.
* `ucapanlebaran-11.html` - `ucapanlebaran-15.html`: Desain Lantern-Shadow & Golden Gateway.
* `ucapanlebaran-16.html` - `ucapanlebaran-20.html`: Cyber Glow Heart & Protective Shield Logic.

## Fitur Utama
- **Interaktif:** Beberapa file memiliki efek partikel cahaya, bintang berkelip, dan animasi melayang.
- **High Readability:** Menggunakan Google Fonts (Montserrat, Playfair, JetBrains Mono) untuk memastikan teks romantis terbaca jelas.
- **Responsive:** Bisa dibuka dengan sempurna melalui perangkat mobile (Smartphone) maupun desktop.
- **Neon-Islamic Aesthetic:** Perpaduan warna khas Mr.Rm19 (Neon Green) dengan nuansa Idul Fitri (Gold/Emas).
